What is Depth Peeling?

Depth Peeling is a method used to render multiple layers of transparency. Unlike simple alpha blending, depth peeling ensures that the overlapping transparent surfaces are accurately visualized and rendered in any order.

  • Left: Displays the object using traditional transparency techniques.
  • Right: Displays the object using depth peeling.

You can change the view angles by mouse dragging and zoom in and out with mouse wheel.
