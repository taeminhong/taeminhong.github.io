Zls Inspector Release Notes

Version 0.3.0

May 14, 2025
  • What's New
    • Now, the app recognizes the following items as well:
      • Zipper handlels
      • Fabrics in the 'infile' folder
      • Stitches in subfolders, such as '0', '1', and so on
      You can test it with these files:
      • M_Jacket_Leather+Pants.zls
      • W_School+Uniform.zls

Version 0.2.2

December 10, 2023
  • Bug Fixes
    • We prevent the contents from being jumbled when attempting to load multiple times in a short period.
    • We hide the border of the history panel when it's empty

Version 0.2.1

October 5, 2023
  • Improvements
    • We enhanced the legibility on mobile devices.
  • Bug Fixes
    • We don't display poses with a uuid of 0/0 anymore.

Version 0.2.0

October 4, 2023
  • What's New
    • Now, we can display artworks and stitches as well.
  • Improvements
    • We now recognize legacy fabrics that use xtex files instead of u3m files
  • Bug Fixes
    • We don't display hidden draping items anymore.

Version 0.1.0

September 30, 2023
  • What's New
    • We'll display the contents below immediately when you drag and drop zls files in the drop zone.
      • Recent edit history
      • Files and directories in a zls file
      • Draping board
      • Poses
    • When you drop a single zls file, we'll also display the contents below immediately. For multiple zls files, simply press the load button for each to display them.
      • Avatars
      • Fabrics
      • Buttons and buttonhols
