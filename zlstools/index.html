What is ZlsTools?

ZlsTools is a Window Explorer extension that generates thumbnail images of zls files so that you can see what's in the files without opening them in an editor. It can help you manage zls files.

Before After

How do I use it?

It's easy. Follow these steps:

  1. Download the installer from here.
  2. Install it.
  3. Open a File Explorer
  4. Go to a folder where zls files exist
  5. From the menu, select "View" > "Large Icons" or scroll up the mouse wheel pressing the Ctrl key.

My zls files still show as icons!

First of all, ZlsTools is still under development. So it may not work on your computer.

Basically, ZlsTools extracts the preview images from the draping boards. So if the file has no draping boards, it will still appear just as an icon. Please, check the file with zlsinspector to see if there is a draping board.
