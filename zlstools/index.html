What is ZlsTools?

ZlsTool is a plugin that generates thumbnail images for zls files so that you can see the contents of a file quickly without opening it in an editor. It can help you manage zls files.

Before After

How do I use it?

It's easy. Follow these steps:

  1. Download the installer from here.
  2. Install it.
  3. Open a File Explorer
  4. Go to the folder where zls files exist
  5. Scroll up the mouse wheel pressing the Control key.

My zls files still show as icons!

First of all, ZlsTools is still under development. So it may not work on your computer.

Basically, ZlsTools seeks zls files and extract the preview image from the draping boards within them. So if the file doesn't have a draping board, thi file will still appear as an icon. Please, check the file with zlsinspector to see if there is a draping board.
